## Conclusions
The top factors that are seen effecting and benefitting the Bike Rental count are as follows:
- year = 0.2359
- temp = 0.3896
- WorkingDay = 0.0238
- September = 0.0667

The top variables that are negatively impacting the sales are
- Light rain_Light snow_Thunderstorm = -0.2884
- windspeed = -0.1445
- Spring = -0.1066
